## üîÑ 5. Uninstalling the Cisco Secure Client

### Standard Method:

- Use the provided **Uninstaller**
  (search via Spotlight or find it in the Cisco Secure Client installation folder)

### If the client was deleted manually:

1. Open Terminal (`Spotlight > "Terminal"`).
2. Run the following commands:

```bash
cd /opt/cisco
```

```bash
sudo find . | grep -i uninstall
```

3. Depending on the result, execute one of the following:

```bash
sudo ./secureclient/bin/cisco_secure_client_uninstall.sh
```

or

```bash
sudo ./anyconnect/bin/anyconnect_uninstall.sh
```

4. Repeat the `find` command and execute any additional uninstall scripts if they appear:

```bash
sudo <output_line>
```
